# Spain DNI Generator

Generate different types of ID number for Spain like NIE, NIF and DNI

## Requirements
 Java 11 + 

## How to use it?

````java
// NIE example
Generator generator = new Generator();
String nie = generator.nie();

// NIF example
String nif = generator.nif();

// CIF Example
String cif = generator.cif();

````
